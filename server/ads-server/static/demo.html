🎯 广告系统演示

智能广告展示与统计系统

欢迎使用广告展示系统

这是一个功能完整的广告展示和统计系统演示页面。系统会自动加载并展示广告，同时统计页面访问量和广告点击量。

🎨 智能展示

自动在页面中心展示主广告，右下角展示次要广告

📊 数据统计

实时统计页面访问量、广告点击量等关键数据

⚙️ 灵活配置

支持广告状态管理、X号重定向等多种配置选项

🔧 易于集成

只需引入一个JS文件即可在任何网站中使用

📝 使用说明

要在您的网站中使用此广告系统，只需在HTML页面中添加以下代码：

<script src="http://localhost:8000/static/ad-script.js"></script>

功能特点：

  • ✅ 自动记录页面访问量
  • ✅ 随机展示主广告和次要广告
  • ✅ 支持广告点击统计
  • ✅ 可配置X号重定向功能
  • ✅ 响应式设计，适配各种设备
